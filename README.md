# ResORR + SWOT

This repository archives the Python notebooks and software developed during the "A Paradigm Shift in Quantifying Regulation of Rivers using Global Water Elevation Measurements" project.

- Notebooks: Contains Python notebooks. Notebooks may become outdated as the research progressed, but the history can be found in the git history if desired.
- src: source code of resorr and time-series smoothening.
- params: config/parameter files used for running RAT.
